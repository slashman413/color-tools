About these free color tools

Everything you need to work with color on the web: pick a color, convert between HEX, RGB and HSL, build a palette, and generate CSS gradients you can copy straight into your stylesheet.

How to use

  1. Pick a color with the picker, or type a color code.
  2. Read the equivalent HEX / RGB / HSL values.
  3. Build a palette or generate a CSS gradient and copy the code.

Why use it

Designers and developers constantly translate between color formats and need quick, copy-ready CSS. Having the picker, converter, palette and gradient generator together speeds up any UI work.

HEX, RGB and HSL, with instant two-way conversion.

Yes — the gradient generator outputs CSS you can paste directly into your stylesheet.

Does it work on mobile?

Yes — it's a responsive browser tool.
